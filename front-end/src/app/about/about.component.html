Acerca de Nuestro Bienestar

Una plataforma creada para mejorar el acceso a programas de salud, deportes, educación y transporte en la universidad.

Esta aplicación frontal se conecta con el backend propio para consumir datos reales desde la API REST. Incluye navegación responsiva, estado de carga, validación de formularios y un diseño que funciona en móviles y escritorio.

  • Frontend construido con Angular 18
  • Consumo de API: GET, POST y DELETE
  • Gestión de sesión básica con JWT
  • Estilos responsivos y navegación persistente
